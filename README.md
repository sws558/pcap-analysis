## 主要功能

+ 1.展示数据包基本信息
+ 2.分析数据包协议
+ 3.分析数据包流量
+ 4.绘制出访问IP经纬度地图
+ 5.提取数据包中特定协议的会话连接（WEB，FTP，Telnet）
+ 6.提取会话中的敏感数据（密码）
+ 7.简单的分析数据包中的安全风险（WEB攻击，暴力破解）
+ 8.提取数据报中的特定协议的传输文件或者所有的二进制文件
